No Column Layout

The No Column Layout uses 100% of the width of its container.

No columns are used when there is no need to divide or separate the content. It's also good for large pages of text, instructions, disclaimers, etc.

The inner class can be used to create padding around the outside of the content.

The box and pad classes can be used to create a border and additional padding.

Here the box and pad classes are wrapped in the inner class, creating even more separation around content.

The box and pad classes are used with the grey class. This is often helpful for 'NOTE' sections or special instructions that need to stand apart from other content.

The box and pad classes are used with the info class. This is often helpful for 'NOTE' or 'INFO' sections or special instructions that distinctly need to stand apart from other content.

The box and pad classes are used with the shadow class to create a border-box effect.
